Project Overview

Project AROC (Amateur Radio Operations Center) is a long-term engineering, documentation, and software development project focused on designing, building, and operating a modern amateur radio communications center.

The project is built around the philosophy that engineering knowledge should be documented with the same care used to build the system itself.

Project AROC combines:

* 44Net networking
* Raspberry Pi servers
* Linux administration
* Web server technologies
* Amateur radio communications
* Emergency communications
* Network engineering
* Systems engineering
* Professional technical documentation

Project Goals

Project AROC has four primary goals:

1. Build a reliable Amateur Radio Operations Center.
2. Create a professional engineering reference library.
3. Develop reusable software and documentation.
4. Preserve engineering knowledge for future expansion.

⸻

Development Philosophy

Project AROC is developed incrementally.

Each work session focuses on a single milestone. At the end of each session, new documentation and supporting files are committed to the repository.

Every document is expected to evolve through multiple revisions as the project matures.
